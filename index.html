How to use

Search

Enter Pokemon card details (name, set code, etc.) and click Fetch Market Data to find cards. Use the checkbox column to select multiple cards, then use the Add button to add selected cards to a tab. Use Clear Sort to reset sorting, and open Columns to control which Search columns are visible.

Tabs

Create custom color-coded tabs to organize your cards. Click the + button to add a new tab. Use the action icons next to the tab name to edit, duplicate, or delete the active tab. Use Move to transfer cards between tabs and Copy to copy a row to your clipboard.

Grid controls

Use Filter rows to narrow visible results, click a column header to sort, and use Clear Sort to return to unsorted order. Drag a header to reorder columns and drag the right edge of a header to resize it. Open Columns to control visible functional and display columns for Search and each tab independently.

Settings

The Theme section lets you choose Auto, Light, or Dark. The functional/display column settings now define default columns for new tabs. Existing Search/tab column visibility is managed in each panel's Columns chooser. You can also configure sources, export fields, search behavior, reset settings, or clear local data.
