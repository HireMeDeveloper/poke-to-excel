How to use

Search

Enter Pokemon card details (name, set code, etc.) and click Fetch Market Data to find cards. Use the checkbox column to select multiple cards, then use the Add button to add selected cards to a tab.

Tabs

Create custom color-coded tabs to organize your cards. Click the + button to add a new tab. Use the Move button to transfer cards between tabs. Tabs are saved automatically in your browser.

Settings

Customize which columns are visible, control which data sources to include in searches, and manage export options. You can also reset settings or clear all data from this page.
